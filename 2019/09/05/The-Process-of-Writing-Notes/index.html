How to write blogs?

Under my blog’s root director：

  1. hexo new post "article title" –> New a .md work;
  2. \source\_posts –> Find the source work, open and recording what you want;
  3. Open the git Bash under my blog’s root director
    1. hexo g生成
    2. hexo s预览
    3. hexo d发布

How to upload my blogs’ pics?
