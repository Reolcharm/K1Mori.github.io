How to write blogs?

Under my blog’s root director：

  1. hexo new post "article title" –> New a .md work;
  2. \source\_posts –> Find the source work, open and recording what you want;
  3. Open the git Bash under my blog’s root director
    1. hexo g生成
    2. hexo s预览
    3. hexo d发布

How to upload my blogs’ pics?

  3. Find the pic what u wanna upload, then use the shortcuts Ctrl + Alt + U. And the pic will upload to u GitPicBed, and the markdown files also paste the pic’s url automatically.

不能忘了那些耻辱和暗无天日的过去。

技术

  • 如果一个人明白了自己能做什么，想做什么，要做什么，那么这个人算是开窍了，在未来的无论工作还是生活，他都能坚强地挺着。有些人开窍早，而那些还没开窍的，终于会被现实开窍，这个现实就是：这个世界不需要你这么菜的菜鸡。

  • 学技术有两个很重要的方法.

    • 第一是工作环境中有很强的项目锤炼，引导、推进你去不断深挖和学习.
    • 第二是自己系统的学习先进的开源软件，如果公司没有这么多的项目用到这些知识，那你就可以通过开源软件去吸纳其他人的思想。
      • 另外，你要有不断提炼和复盘的时间，写代码的人看自己两年前的代码跟屎一样，这是一个段子，也是真实的情况，你会不断发现之前有一些错误，不断进步。
      • 但是，很关键的一点是这个过程中一定要有提炼，融会贯通，才能在知道已知的问题后，从容应对未知的问题。

  • 老师或父母老是说，努力就会走到巅峰—-才怪。如果这样，不是所有人都走上巅峰了吗？没有人开始不努力，为什么后来不努力，因为努力没有效果。”人生不是走斜坡，你持续走就可以走到巅峰；

    • 人生像走阶梯，每一阶有每一阶的难点，学物理有物理的难点，学漫画有漫画的难点，你没有克服难点，再怎么努力都是原地跳。所以当你克服难点，你跳上去就不会下来了。蔡志忠的核心观点就是黑体的那句话，成功的人生是台阶式向上，而不是一条水平线。努力只是说明你拼命在走，跟你能不能向上走，关系不大。
    • 那些努力却没有结果的人，根本原因就在于，他一直走在平面上，没有走到更高的台阶。也就是说，垂直方向的努力更有意义，水平方向的努力意义不大。你把同一件事情勤奋地做上十遍，还是只会做这一件事；你做完这件事后，再去挑战更难的事情，就有机会学会做两件事。

  • 沉迷撸代码，实现各种功能、探寻实现原理。

    • 真正将敲代码当作一件开心，快乐的事，“我能实现这个功能”。给予自己这样的心里暗示。
    • 主动挑战自己的大脑，锻炼自己的大脑。有道是学无涯而生有涯也，不断停止探寻的脚步

  • 我们不是不喜欢培训的学生，而是不喜欢不思考停步不前的学生，

    • 因为现在的很多开发框架都是成熟的，一个脑袋稍微不差的人，只要经常去敲，去做。他都能胜任工作，但是他们对于底层的理解很少，尽管很少有人在开发，尤其是现在的web开发当中很少用到底层，但因为缺乏这一层的概念理解，他们很难成为一个优秀的程序猿~

  • 我跟你讲讲我的故事吧。大学毕业，不知道干什么所以报了个网页设计培训班。培训班费用不高，我还和她们砍了500元价格，总共也就四千块。班里面人员很复杂什么行业的都有，学生，商人…所以课程必须从相当基础开始，但是课期只要四个月，所以可想而知学员的水平。后来找工作的时候，大部分优秀公司是不要的，去了一家小不啦叽的公司写网站，工作内容无非就是去偷拔别人的代码，毛都没学到。一年后公司垮了，然后再找相关工作，不甘心工资仍然是一年前水平。愤怒转行做销售，到现在25岁一事无成，无钱，无房，无车，无女朋友。混成这个卵样，有一些偶然性，也有一些必然性。

    • 但是我认为对我起到最终影响的是三点，第一缺乏明确的目标和职业规划，第二缺乏刻苦专研的自学态度，第三缺乏一直做下去的兴趣毅力。但是话又说回来，如果具备这三样，那报不报培训班对我又有什么影响呢？

Two Tips

Share

1. 提高自己的起点

  • 当别人都在看视频的时候，我坚持看博客、文档学习，因为看视频很没效率，看博客除了节省我的学习时间之外，还提高了我的阅读能力。
  • 当别人在用百度，查看国内各种二手资料的时候，我坚持使用 Google，查看国外大牛博客，我英语很差，这对我来说很难，但是我硬逼着自己，坚持下去;
  • 当别人混国内小白论坛的时候，我就开始混 GitHub 开源社区，学习国外大牛的开源项目，当别人在看国内翻译的资料的时候，我却逼着自己去官方看英文文档。

这一切的一切，本质上都是为了提高自己的起点，虽然我起步晚，但是我起点高，慢慢的就会缩小差距，时间长了，甚至后来居上。 当然，前提是自己得衡量自己的能力，就好比，你才刚学会写一个 Hello World ，你说你要去看 Java 源码，这不现实。

所以，我在最初入门的时候也经历过看视频，百度的阶段，但是一旦自己上手之后，我坚定的逼着自己让自己的起点更高。

2. 遇事多思考

  1. 哪怕一开始是错误的，但是也不要多停止思考，只有多思考，才能提升自己的辨别力；
  2. 信息泛滥的时代，其实有一点非常重要，就是要找到你信任的、比你厉害的人，多看他们的观点与推荐，这不是在说我，我自己也有一些我信任的，比我厉害的人，这会让自己少走弯路，多专注在重要的信息上。

多说一句，其实，学习在这个时代，在我看来，这一点都异常重要，很多人的学习方式都还停留在以往靠自己，在海量信息中去寻找去摸索的时代，这种方式在以往信息匮乏的年代固然好，但是在信息泛滥的时代，这种学习方式是低效的, 在我看来，当下的学习，同样要多借助比你厉害的人的推荐，不管是书籍也好，还是学习方法等，那些比你厉害的人的推荐，能让你少做一些无用功。
